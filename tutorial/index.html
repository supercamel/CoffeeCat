Hello World

Here's the mandatory hello world example

int main():
    print("Hello world!\n")

This is just one function call main and all it does is print out a famous message. CoffeeCat uses indentation to specify blocks of code. Indents can be either 4 spaces or a tab; the compiler identifies both of these as an indent.

The CoffeeCat compiler (let's call it ccat from here on) turns this source into C++. To compile, open a terminal and use 

ccat -o ./ hello.cof

If all goes well, you will see the files hello.cpp, hello.h and coffee_header.h appear.

To compile the C++ files, a couple of dependancies are required. The first is ETK, which can be cloned from github 

git clone http://github.com/supercamel/EmbeddedToolKit
cd EmbeddedToolKit
make && make install

The second dependancy is a small 'standard' library which CoffeeCat needs to print on to the terminal. This can be copied from the examples directory (link) and it should be placed in the same folder as your hello.cof file.

To compile, use 

g++ -std=c++14 hello.cpp stdlib/print.cpp -letk -o hello

Flow Control

CoffeeCat has only four control statements. These are

  • if
  • else if
  • while
  • for

It also has break and continue statements that can be used inside while and for loops.

If statements are used to check for true conditions. 

if(a == 5):
    print("a is equal to five")
